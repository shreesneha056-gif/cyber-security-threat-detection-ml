---

## 📌 Problem Statement
Cyber attacks are growing in frequency and complexity. This project builds ML models to automatically detect and categorize network intrusions from traffic telemetry data, enabling faster threat response.

## 🎯 Key Results
| Task | Model | Description |
|------|-------|-------------|
| Binary Classification | ML Classifier | Normal vs Attack traffic |
| Multiclass Classification | ML Classifier | Attack type categorization |

## 📊 Pipeline Overview
1. **Data Loading** — Load network traffic telemetry dataset
2. **EDA** — Explore feature distributions and class imbalances
3. **Preprocessing** — Encode categorical features, scale numerics
4. **Binary Classification** — Normal vs Attack detection
5. **Multiclass Classification** — Identify specific attack types
6. **Evaluation** — Accuracy, precision, recall, F1-score, confusion matrix
